## Installation

### Visual Studio Marketplace

Paste the following command and press `Enter`:

```shell
ext install canducci.canducci-reactjs-snippets
```
-------------------------------------------------------------------------------------------------------

## Supported languages (file extensions)

- JavaScript (.js)
- JavaScript React (.jsx)
-------------------------------------------------------------------------------------------------------
